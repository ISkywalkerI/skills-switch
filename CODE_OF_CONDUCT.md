# Code of Conduct

## Our Pledge

We want Skills Switch to be an open, respectful, and welcoming project for everyone.

## Expected Behavior

- Be respectful and constructive
- Assume good intent, but communicate clearly
- Focus feedback on the work, not the person
- Be patient with questions from new contributors
- Accept and act on reasonable feedback gracefully

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory language or behavior
- Trolling, intimidation, or sustained hostility
- Publishing private information without permission
- Any other conduct that would be inappropriate in a professional setting

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that violate this Code of Conduct.

## Reporting

To report unacceptable behavior, open a private security-style report if possible or contact the maintainer directly through the contact method listed in `SECURITY.md`.

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, discussions, and other public community interactions.
